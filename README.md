# TaskFlow

TaskFlow is a project management application designed to help teams organize projects, create tasks, assign work, track progress, and communicate through task comments.

## Features

- User registration and login
- Create and manage projects
- Create tasks inside projects
- Assign tasks to users
- Track tasks using:
  - To Do
  - In Progress
  - Completed
- Add comments to tasks
- Delete tasks
- Responsive dashboard interface

## Technologies Used

### Frontend
- React
- Vite
- CSS

### Backend
- Python
- Flask
- SQLite
- SQLAlchemy
